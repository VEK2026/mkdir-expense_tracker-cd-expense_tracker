## Описание
Простое десктопное приложение для учета личных расходов. Позволяет добавлять расходы, просматривать их в таблице, фильтровать по дате/категории и сохранять данные в JSON файл.

## Как использовать
1. Запустите скрипт: `python main.py`
2. **Добавление:** Заполните Сумму, выберите категорию, укажите дату (ГГГГ-ММ-ДД) и нажмите "Добавить".
3. **Фильтрация:** Введите дату "С" и "По" или выберите категорию, нажмите "Фильтр".
4. **Сброс:** Нажмите "Сброс", чтобы увидеть все данные.

## Технические особенности
- Валидация: Сумма должна быть числом > 0, дата в формате ГГГГ-ММ-ДД.
- Сохранение: Данные хранятся в файле `data.json`.
